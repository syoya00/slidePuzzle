・操作

slidePuzzle
Enter:ゲーム開始（スタート画面）、名前の決定（名前入力画面）
カーソルキー上下:アルファベットの選択（名前入力画面）
カーソルキー左右：文字箇所の選択（名前入力画面）
マウス左クリック：パズルをつかむ＆離す（ゲーム画面）
マウス右クリック：つかんだパズルの移動（ゲーム画面）
パズルを並べて高得点を目指しましょう。
縦か横に3つ以上同じ色のパズルがそろうと消えます。
パズルが消えた後に落ちてきたパズルがそろうと連鎖して消えます。
パズルを連鎖して消すと高得点が入るのでどんどん連鎖しましょう。
3文字のアルファベットを入力してランキングに登録することもできます。
